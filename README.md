# NeuroStage-ProcessingPipeline
NeuroStage is a fully automated neuroimaging pipeline that takes raw DICOMs through BIDS conversion, quality control (MRIQC), structural reconstruction (FreeSurfer + HCP Pipelines), diffusion MRI (QSIPrep), functional MRI (fMRIPrep), and ASL perfusion (ASLPrep) preprocessing, with automatic dependency checks, crash, and parallel execution support.
